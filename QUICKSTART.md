# Быстрый старт

## Что уже реализовано:

✅ Главное меню с кнопкой "Играть"
✅ Игрок с движением влево-вправо (свайп)
✅ Автоматическая стрельба
✅ Враги, движущиеся по 3 рядам
✅ Система волн (10 врагов на волну)
✅ Ворота усиления (+1, +2, -1)
✅ Принцесса в оковах (нужно стрелять, чтобы освободить)
✅ Босс (появляется на 3 волне)
✅ Коллизии снарядов с врагами и боссом

## Как запустить:

### Вариант 1: Android (рекомендуется для начала)

1. Подключите Android устройство или запустите эмулятор
2. Откройте терминал в папке `dream_defense`
3. Выполните:
```bash
flutter run
```

### Вариант 2: Web (быстрый тест)

```bash
flutter run -d chrome
```

### Вариант 3: Windows

Сначала включите режим разработчика:
1. Нажмите Win+I (Настройки)
2. Перейдите в "Конфиденциальность и безопасность" → "Для разработчиков"
3. Включите "Режим разработчика"
4. Затем:
```bash
flutter run -d windows
```

## Управление:

- **Движение**: Зажмите и двигайте пальцем/мышью влево-вправо
- **Стрельба**: Автоматическая
- **Цель**: Собирайте ворота с +1/+2, избегайте -1, убейте босса или освободите принцессу

## Следующие шаги:

1. Запустите игру и протестируйте базовый геймплей
2. Настройте баланс в `lib/game/dream_defense_game.dart`
3. Добавьте графику (замените placeholder фигуры на спрайты)
4. Добавьте звуки через flame_audio
5. Создайте экраны победы/поражения

## Проблемы?

- **Ошибка symlink на Windows**: Включите режим разработчика (см. выше)
- **Нет устройств**: Запустите Android эмулятор или используйте Chrome
- **Ошибки компиляции**: Выполните `flutter clean` и `flutter pub get`
