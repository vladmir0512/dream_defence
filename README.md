# Dream Defense - Казуальная игра на Flutter

## Описание игры

Dream Defense - это казуальная мобильная игра в жанре tower defense с элементами аркады. Игрок управляет героем, который защищается от волн монстров, движущихся по трем рядам.

### Основные механики:

1. **Управление**: Свайп влево-вправо для перемещения героя
2. **Стрельба**: Автоматическая стрельба вверх
3. **Враги**: Движутся вниз по 3 рядам (левый, центральный, правый)
4. **Усиления**: Прозрачные ворота с надписями +1, +2, -1 (увеличивают/уменьшают количество героев)
5. **Принцесса**: Эльфийка в оковах вверху экрана - нужно стрелять по оковам, чтобы освободить её
6. **Босс**: Появляется на 3 волне, очень сильный противник
7. **Волны**: 10 врагов на волну, после уничтожения всех начинается следующая волна

### Цель игры:

- Усилиться, собирая бонусы (+1, +2)
- Убить босса ИЛИ освободить принцессу раньше
- Выбор стратегии: фокус на врагах или на спасении принцессы

## Технический стек

## Установка и запуск

### Требования:

- Flutter SDK 3.0+
- Android Studio / Xcode (для мобильных платформ)
- Режим разработчика Windows (для запуска на Windows)

### Шаги:

1. Клонируйте репозиторий или перейдите в папку проекта:
```bash
cd dream_defense
```

2. Установите зависимости:
```bash
flutter pub get
```

3. Включите режим разработчика Windows (если запускаете на Windows):
```bash
start ms-settings:developers
```
Включите "Режим разработчика" в настройках.

4. Запустите игру:

**На Android эмуляторе/устройстве:**
```bash
flutter run
```

**На iOS симуляторе/устройстве:**
```bash
flutter run -d ios
```

**На Windows:**
```bash
flutter run -d windows
```

**На Web:**
```bash
flutter run -d chrome
```

## Игровые компоненты

### Player (Игрок)
- Движение влево-вправо по нижней части экрана
- Автоматическая стрельба
- Количество героев влияет на количество снарядов

### Enemy (Враг)
- 3 типа по цвету (красный, оранжевый, фиолетовый) в зависимости от ряда
- 3 единицы здоровья
- Движутся вниз к игроку

### PowerUpGate (Ворота усиления)
- +1: 70% шанс появления
- +2: 20% шанс появления
- -1: 10% шанс появления
- Полупрозрачные стены с текстом

### Princess (Принцесса)
- Находится вверху экрана
- 100 единиц здоровья оков
- Каждый выстрел по оковам уменьшает их прочность
- При освобождении - победа

### Boss (Босс)
- Появляется на 3 волне
- 50 единиц здоровья
- Большой размер (80x100)
- Медленное движение
